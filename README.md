# goread
A self-hosted server and web client for managing and reading e-books and manga/comics.

## Features
- **Self-Hosted**: Run the server on your own hardware or cloud service.
- **Web Client**: Access your library from any device with a web browser.
- **User Management**: Create multiple user accounts with different access levels.
- **Reading Progress Sync**: Keep track of your reading progress across devices.
- **Metadata Management**: Edit and manage metadata for your e-books and manga/comics.
- **Notes and Quotes**: Highlight text and add notes while reading.
- **Dictionary and Personages Lookup**: Look up words and characters directly from the reader.
- **Export data**: Export your library data in various formats for backup or migration.
- **Webhooks**: Integrate with other services using webhooks for events like new book additions.

and none of this is implemented yet. :)
